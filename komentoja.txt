Uusi node projekti / New node project

Luo hakemisto projektin nimellä.
Hakemistossa aja komento "npm init"
Vastaa kysymyksiin, erityisesti entry point ("server.js");

Create a folder with the name of the project
In folder, run "npm init"
Answer questions especially entry point ("server.js");

Kirjastojen installointi / Library installation

Projektihakemistossa (sama kuin missä package.json sijaitsee) aja
npm install <kirjaston nimi> (esim. npm install express)

In project folder (same as where package.json is) run
npm install <library name> (fex. npm install express);
